Projeto

O projeto foi feito utilizando a blockchain Ethereum, o framework Truffle e as ferramentas Ganache e MetaMask. O Truffle disponibiliza um ambiente de desenvolvimento prático para projetos baseados em blockchain e utiliza a API Web3.js, facilitando a criação e integração dos contratos inteligentes desenvolvidos em Solidity com o frontend das aplicações. O Ganache após instalado oferece dez contas de desenvolvimento na rede Ethereum para os testes do projeto e o MetaMask por sua vez se encarrega de fazer a comunicação local dessas contas com o frontend da aplicação. Acesse as informações técnicas e o código do projeto neste link.


Multiverse Pass

O Multiverse Pass é um sistema de teleporte que tranfere os soldados do Planeta Terra para o New Universe, um universo ainda inexplorado pronto para ser conquistado pelos soldados mais capazes. As classes são definidas de acordo com as características de cada soldado e podem ser Guerreiro, Mago, Sacerdote, Arqueiro, Bárbaro, Feiticeira, Espiritualista e Mercenário.

A partir do momento que o soldado realiza a viagem para o New Universe, ele disputa com os outros pelos melhores recursos disponíveis. Que eles tenham uma grande jornada!
